# cad-viewer



## About The Project

This project is made mainly for my personal use, but I would be glad if it helps you too!



## Features

- display cad entities in **2D**
- support simple conctrols, e.g. pan, zoom, copy, paste, ...
- data export and import **(as json)**
- and so on
